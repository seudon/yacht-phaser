# Yacht Dice

Phaser 3 と `@3d-dice/dice-box` で作った、スマホブラウザ向けの一人用ヨットです。

## 遊び方

- 1ラウンドにつき最大3回までサイコロを振れます。
- 出目をタップするとキープできます。
- スコアシートの未使用カテゴリをタップすると、その役で得点を確定します。
- 13カテゴリをすべて埋めるとゲーム終了です。
- 上段合計が63点以上なら35点ボーナスが入ります。

## 開発

```bash
npm install
npm run dev
```

dice-box の配布ファイルは `vendor/dice-box` に、実行時アセットは `public/assets` に配置しています。
